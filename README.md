# ShowcaseCarousel
Componente web para desplegar contenido multimedia.

## Desarrollo

### Requerimientos
* NodeJS

### Instalación
```
npm install
```
### Ejecución
#### Desarrollo
```
npm run serve
```
#### Distribución
```
npm run build
```


```
<!-- file: demo.html -->
<script src=`${PATH_TO_SHOWCASE_SRC}/dist/index.js`>

<showcase-carousel>
  <img src="lorempicsum.jpg">
</showcase-carousel>

```
